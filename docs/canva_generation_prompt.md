# Canva 在线设计生成提示

## 使用方式

把下面的提示复制到 Canva Magic Design、Canva 演示生成或 Canva 设计助手中，用于生成 AI-English-Coach 的在线展示稿。当前 Codex 会话没有暴露 Canva 创建/编辑 MCP 工具，因此本文件用于最小化手动复刻成本。

## 推荐提示

```text
请创建一个 1440 x 1000 的比赛项目展示海报，主题是“AI 英语口语陪练 AI-English-Coach”。

设计风格：
- 深色科技风
- 专业教育工具
- 清晰、现代、适合比赛展示截图
- 不要使用花哨插画，不要使用浅色大面积背景
- 使用卡片式信息分区

颜色：
- 页面背景：#0b1220
- 主卡片：#101827
- 次级卡片：#111f32
- 边框：#243244 / #334155
- 主文字：#f8fafc
- 次级文字：#cbd5e1
- 强调蓝：#38bdf8
- 完成绿色：#22c55e
- 主操作红：#ff4b4b

画面结构：
1. 顶部左侧：
   - 小字：AI-ENGLISH-COACH PRODUCT DESIGN
   - 大标题：AI 英语口语陪练
   - 副标题：面向比赛展示的本地可运行英语口语练习工具：场景对话、纠错反馈、五维评分、课后总结和历史导出一屏呈现。

2. 顶部右侧四个状态卡：
   - 视觉设计：深色科技风
   - 交互设计：7 步练习流
   - 信息架构：5 个标签页
   - 运行模式：API / 本地演示

3. 中部左侧“核心用户流程”：
   - 1 选择练习场景：面试、点餐、会议，对应不同 AI 角色和表达目标。
   - 2 输入或上传音频：支持文本输入和音频转写降级，不阻断主流程。
   - 3 查看 AI 回复：根据场景追问，让用户形成连续对话训练。
   - 4 获得纠错与评分：语法、表达、跟练句和五维评分清晰分区。
   - 5 导出课后报告：Markdown / HTML 报告和 JSON 历史记录可保存。

4. 中部右侧放一个简化版产品界面 mockup：
   - 顶部卡片：AI 英语口语陪练
   - 状态文字：模式：本地演示模式；场景：面试 Interview；难度：中等；进度：25%
   - 左侧卡片：对话练习、当前练习任务：Project 项目经历、输入框、提交练习按钮、生成总结按钮
   - 右侧卡片：AI 回复、评分、综合评分 86/100、蓝绿进度条

5. 底部四个指标卡：
   - 3 练习场景
   - 5 评分维度
   - JSON 历史记录保存
   - MD / HTML 课后总结导出

排版要求：
- 画面留白充足
- 标题大而清晰
- 卡片边框细，不要过粗
- 评分条要清楚可见
- 所有文字必须可读，不要白底白字
- 整体像一个真实 AI 学习工具的产品展示页
```
